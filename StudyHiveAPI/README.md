# StudyHive API

A full-stack study material sharing backend built using ASP.NET Core Web API.

---

# Features

## Authentication
- User Registration
- User Login
- JWT Authentication

## Materials
- Upload Materials
- Get All Materials
- Get Material By Id
- Update Material
- Delete Material
- Download Material
- Download Tracking

## Ratings
- Add Rating
- Update Rating
- Get Ratings

## Bookmarks
- Add Bookmark
- Remove Bookmark
- Get My Bookmarks

## Profile
- Get My Profile
- My Uploads
- My Downloads

## Advanced Features
- Pagination
- Sorting
- Search
- DTO Validation
- File Validation
- Global Exception Handling

---

# Tech Stack

- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- JWT Authentication
- Swagger

---

# Run Project

1. Clone repository
2. Update connection string in appsettings.json
3. Run migrations
4. Start project
