# StudyHive Frontend

Frontend application for **StudyHive** — a collaborative platform where students can upload, explore, bookmark, rate, and download study materials.

Built using **Flutter** with Provider state management and Dio for API integration.

---

## Features

- User Authentication (JWT)
- Material Upload System
- Download Study Materials
- Bookmark Materials
- Rating System
- User Profile
- My Uploads & Downloads
- Clean Modern UI
- REST API Integration

---

## Tech Stack

- Flutter
- Dart
- Provider
- Dio
- ASP.NET Core Web API (Backend)
